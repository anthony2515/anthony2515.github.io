Defining Variables

Defining variables in Javascript is easier using let,var, const but this three have different purposes on declaring values.


  • var keyword
    • are function-scoped or globally scoped, depending on where they are declared.
    • This means they are accessible throughout the entire function or in the global scope if declared outside of any function.
  • let keyword
    • are block-scoped. They are only accessible within the block (enclosed by curly braces) in which they are defined.
    • Block scope includes loops, conditional statements, and any other code blocks.
  • const keyword
    • are intended to hold constant or immutable values.
    • Once you assign a value to a const variable, you cannot reassign it to a different value.

Control Flow

Control Flow is a fundamental concept in programming that refers to the order in which statements and instructions are executed in a program. So basically, when we break down the word control and flow, control means you can control whatever instruction you tell to the computer and flow is like the flow of your instruction in order for the computer to understand what you are saying to it. So we have different ways to use control flow for the computer to understand what we wanna tell them. And those control flows are:


  • Condition Statements
  • Loops
  • Function Calls

DOM(Document Object Model)

The DOM or Document Object Model, is a programming interface for web documents.


It represents the page so that programs can change the document structure, style, and content dynamically.The DOM is a way for programs (typically written in JavaScript) to interact with web pages and make them dynamic and interactive.
